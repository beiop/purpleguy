# purpleguy
this will teleport you back to 0,0,0, feel free to edit main.py to change these coordinates to whatever you want to teleport back to.



## to install ig:
```bash
rm -rf purpleguy-main && wget https://github.com/beiop/purpleguy/archive/refs/heads/main.zip && unzip main.zip && rm main.zip && cd purpleguy-main && bash pre.bash && cd ..
```

## to run ig:
```bash
purpleguy-main/.venv/bin/python purpleguy-main/main.py
```
