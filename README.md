# purpleguy
this will teleport you back to 0,0,0, feel free to edit main.py to change these coordinates to whatever you want to teleport back to.



## to install ig:
```bash
wget https://github.com/beiop/purpleguy/archive/refs/heads/main.zip && unzip main.zip && rm main.zip && bash purpleguy-main/pre.bash
```

## to run ig:
```bash
python3 purpleguy-main/main.py
```
